# Legs
## 项目介绍

校园订单系统 校园接单 小哥接单 客服放单 管理员管理界面
软件使用IDea WebStorm

## 文件树
~~~ txt
Legs
├─front 新版前端项目文件夹
├─old 旧版前后项目文件夹
│  ├─src 旧版 java代码
│  └─web 旧版 前端代码
└─server 新版后端端项目文件夹
└─leg-sql.sql    mysql 建表语句
~~~
